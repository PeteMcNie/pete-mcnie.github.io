Positioning

Differences between absolute, relastive and fixed postioning

Position: Absolute;

Positioning absolute is a powerful tool but should be used with caution if you don't know what you're doing...like me. Every HTML document is as a block flow of elements going down the page. By assigning position: absolute; to an item we arre removing it from the block flow. This means other items will ignore it and move up as if it was not there. Usually you will see the position absolute item now covering these items that have moved up behind it. Position: absolute; items look for a parent element that also have a position element. Let's look at two cases: Firstly, no postion element anywhere else - our position: absolute; element will be positioned with regards to the html element you have set when you first created your page. Great if you want your element to remain somewhere like the top left corner of the document. Maybe a logo. Secondly, if a parent element has a position element - the position: absolute; element will be positioned with regards to this parent element. Great if you want something to remain postioned in the top left of that parent element.
